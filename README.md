#  AI Mood Tracker

An AI-powered Mood Tracker built with React, TypeScript, Vite, and Tailwind CSS that helps users monitor their emotional well-being through daily mood logging, insightful analytics, and personalized AI-powered recommendations.

## Features

-  Daily mood logging
- Mood history tracking
-  Interactive mood analytics
-  AI-generated mood insights
-  Dark & Light mode support
-  Fully responsive UI
-  Fast performance with Vite
-  Modern interface built with Tailwind CSS

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/hemanthreddy16/Ai-Mood-Tracker.git
```

Navigate to the project

```bash
cd Ai-Mood-Tracker
```

Install dependencies

Using npm

```bash
npm install
```

or using Bun

```bash
bun install
```

---

## ▶️ Run the Project

Using npm

```bash
npm run dev
```

Using Bun

```bash
bun run dev
```

Open your browser and visit

```
http://localhost:5173
```

## 🎯 How It Works

1. User logs their mood.
2. The application stores the mood history.
3. AI analyzes mood trends.
4. Personalized wellness suggestions are generated.
5. Dashboard displays emotional progress over time.
